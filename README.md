# 江苏好运科技有限公司 - 官方网站

这是一个为江苏好运科技有限公司设计的企业官网项目。该网站采用现代化的简约设计风格，具备多页面切换、动态轮播图、响应式布局以及开源地图集成等功能。

## 🌐 访问地址
- 远程仓库: `https://github.com/brownchow/companyHomePage.git`

## ✨ 功能特点

- **简约美观的设计**：采用蓝青渐变配色，整体风格专业且具有科技感，解决了页面“太空”和配色单调的问题。
- **多页面切换**：独立的 HTML 页面架构（首页、产品中心、关于我们、新闻动态、联系我们），支持 Tab 切换，而非单页滚动。
- **动态轮播图**：
    - 首页顶部包含 3 张高清科技感轮播图。
    - 支持自动循环播放（5秒间隔）。
    - 支持用户通过下方指示器（小白点）手动点击切换。
    - 鼠标悬停时自动暂停，离开后继续播放。
- **新闻详情系统**：
    - 专门的 `news/` 目录存放新闻详情页，保持根目录整洁。
    - 包含 3 篇预填的科技感新闻详情。
- **开源地图集成**：
    - 在“联系我们”页面集成了 **Leaflet + OpenStreetMap**。
    - 无需任何商业 API Key（如百度地图 AK）即可直接展示公司准确位置。
- **响应式布局**：完美兼容手机、平板及电脑等不同尺寸的屏幕设备。

## 🛠️ 使用技术

- **HTML5**：页面结构化布局。
- **CSS3**：使用 Flexbox、Grid 布局，以及现代渐变和动画效果。
- **JavaScript (Vanilla JS)**：实现轮播图逻辑、导航交互及地图集成。
- **Font Awesome**：丰富的矢量图标支持。
- **Leaflet**：轻量级开源地图库。

## 🚀 如何使用

1. **克隆项目**：
   ```bash
   git clone https://github.com/brownchow/companyHomePage.git
   ```
2. **本地预览**：
   直接使用浏览器打开根目录下的 `index.html` 文件即可查看完整效果。
3. **部署**：
   由于是纯静态页面项目，您可以直接将所有文件上传至 GitHub Pages、Vercel、Netlify 或任何支持静态托管的 Web 服务器上。
